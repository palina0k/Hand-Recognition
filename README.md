# Hand and Mouth Detection with Hamster Reactions

A fun Python project that uses computer vision to detect hand gestures and mouth movements, displaying different hamster images based on the detected state.

## Features
- Real-time hand detection and finger counting
- Mouth open/closed detection
- Displays different hamster images based on gestures

## Steup
1. Clone the repository
2. Install dependancies: `pip install opencv-python mediapipe`
3. Run: `python hamster.py`

## Controls
- 'q' to quit
- Show peach sign (or any 2 finger combination) for peace hamster
- Open mouth for scared hamster
- Default shows normal hamster

## Requirements
- Python 3.8+
- OpenCV
- MediaPipe
